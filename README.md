# Ash's Portfolio

A modern, responsive portfolio showcasing my work at the intersection of full-stack development and creative design. Built to demonstrate both technical execution and user-centered thinking.

**[Live Site](ashleyarciniega.com)** | **[LinkedIn](https://www.linkedin.com/in/ashley-arciniega/)** | **[Email](arcinie3@msu.edu)**

## About This Project

This portfolio is more than just a showcase - it's a working example of my approach to building user-focused web applications. I designed and developed this site to be fast, accessible, and thoughtfully crafted from the ground up.

### What I Built

- **Responsive Design System** - Mobile-first approach ensuring seamless experience across all devices
- **Project Showcases** - Detailed case studies of my development work, including the HAP AI Training Platform and UI/UX designs
- **Creative Section** - Illustration work and video content demonstrating visual storytelling and design thinking
- **Dynamic Resume System** - Database-driven resume with automated CSV import and normalized schema
- **Performance Optimized** - Fast load times and smooth interactions throughout

## Tech Stack

**Frontend**
- React - Component-based architecture for maintainable, scalable UI
- Responsive design with mobile-first methodology

**Backend & Data**
- Flask - Python web framework powering the backend
- MySQL - Normalized database with 4 related tables for resume data management
- Custom Python data models for automated CSV import system

**Deployment & Infrastructure**
- Google Cloud Platform - Scalable, production-ready hosting
- Docker - Containerized deployment for consistency across environments

**Development Tools**
- Git & GitHub - Version control
- VS Code - Development environment

## Running Locally

Want to see how it's built? Clone and run:
```bash
git clone https://github.com/asharcinie3/portfolio-website.git
cd portfolio-website
docker-compose up --build
```

Then open `http://localhost:8080` in your browser.

## Deployment

This site is containerized with Docker and deployed on Google Cloud Platform for reliability and scalability.
